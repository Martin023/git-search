Hello, Welcome to Git-search......

Welcome to git-search. Here we make it easy for you to see the profile and public repositories of any user. All you need is their Github user names or repository names and you will be good to go. You feed in that user name in the search bar below and in a press of a button, you will have their full profile.

Are you new to GitHub?

Github is a Git repository hosting service, but it adds many of its own features. While Git is a command line tool, GitHub provides a Web-based graphical interface. It also provides access control and several collaboration features, such as a wikis and basic task management tools for every project.

How to use the app :

To search for any user, navigate to git-search section on the navigation bar and you will be able to search for any repo or user.
